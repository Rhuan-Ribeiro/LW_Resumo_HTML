Aula de Linguagem Web - HTML

1. Estrutura Básica do HTML

Metatags

As metatags ficam dentro do <head> e servem para configurar o documento.

  • charset → Define a codificação de caracteres.
  • description → Descrição da página
  • keywords → Palavras-chave
  • author → Autor do site
  • viewport → Responsividade

Comentários

Comentários não aparecem na página. Exemplo:

<!-- Isto é um comentário -->

Atributos e Especificações

Os atributos ficam dentro das tags e fornecem informações extras.

  • href → usado em links
  • src → usado em imagens
  • id → identificador único
  • title → texto explicativo

2. Definição dos Conceitos de HTML

HTML é uma linguagem de marcação utilizada para publicação de documentos na web.

Comandos (Tags)

Exemplos de tags:

  • <p> → Parágrafo
  • <br> → Quebra de linha
  • <hr> → Linha horizontal
  • <div> → Divisão de conteúdo
  • <span> → Agrupa conteúdo em linha (inline)
