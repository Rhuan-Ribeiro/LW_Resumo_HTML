Sobre esta página

Esta página foi criada para servir como exemplo em uma aula de CSS básico. Ela possui textos, botões, caixas, links e seções organizadas para facilitar a aplicação de estilos.

O objetivo é observar como o CSS modifica a aparência de uma página HTML por meio de seletores, propriedades e valores.
