# 📘 Aula de HTML – Projeto Introdutório

Este projeto foi desenvolvido para a disciplina de **Linguagem Web** com o objetivo de apresentar, na prática, os principais conceitos do HTML.

Além de servir como exercício, este material também funciona como um **resumo para revisão** dos conteúdos do primeiro bimestre.

---

## 🎯 Objetivos

- Demonstrar a estrutura básica de um documento HTML
- Apresentar as principais tags utilizadas
- Mostrar exemplos práticos de formatação de texto
- Trabalhar com listas e links
- Servir como material de apoio para estudos

---

## 🧱 Conteúdos abordados

### 📌 Estrutura básica
- `<!DOCTYPE html>`
- `<html>`, `<head>` e `<body>`
- Metatags

### 🏷️ Tags principais
- Parágrafos (`<p>`)
- Quebra de linha (`<br>`)
- Linha horizontal (`<hr>`)
- Divisões (`<div>`)

### ✍️ Formatação de texto
- Negrito (`<strong>`)
- Itálico (`<em>`)
- Sublinhado (`<u>`)
- Destaque (`<mark>`)
- Texto pequeno (`<small>`)

### 🔢 Listas
- Lista não ordenada (`<ul>`)
- Lista ordenada (`<ol>`)
- Lista de definição (`<dl>`)

### 📷 Imagens
- Caminho da Imagem (`src=""`)
- Descrição da Imagem (`alt=""`)

### 🔗 Links
- Links externos
- Links internos (âncoras)

### ⚙️ Atributos
- `href`
- `src`
- `id`
- `title`

---

## ⚠️ Observações

- Algumas práticas mostradas (como o uso da tag `<font>`) são **obsoletas no HTML5**, mas foram incluídas para fins didáticos.
- Em projetos reais, recomenda-se utilizar **CSS** para estilização.
